🌍 World Layoffs Analysis Dashboard (Power BI)

📌 Project Overview

This project analyzes global layoff trends using Power BI. The dashboard provides insights into layoffs across different companies, industries, countries, and years, helping users understand workforce reduction patterns and identify key trends.

🎯 Objectives

- Analyze total layoffs worldwide.
- Compare layoffs across companies and industries.
- Identify the countries with the highest layoffs.
- Track layoff trends over time.
- Provide interactive filtering for better analysis.

🛠️ Tools Used

- Power BI
- Power Query
- DAX
- Microsoft Excel / SQL (Data Source)

📊 Dashboard Features

- Total Layoffs KPI
- Layoffs by Company
- Layoffs by Industry
- Layoffs by Country
- Year-wise & Month-wise Trend Analysis
- Interactive Slicers and Filters
- Dynamic Visualizations

📈 Key Insights

- Identifies companies with the highest layoffs.
- Shows industries most affected by workforce reductions.
- Compares layoffs across different countries.
- Displays layoff trends over multiple years.
- Helps understand global employment patterns.

📂 Files Included

- "World_layoffs.pbix" – Power BI Dashboard
- "README.md" – Project Documentation

🚀 How to Use

1. Download the ".pbix" file.
2. Open it using Microsoft Power BI Desktop.
3. Refresh the dataset (if applicable).
4. Explore the dashboard using filters and slicers.
